Assignment

Here are some fun practice activities for you

Activity - 1

In the year 2020 everybody was going through the pandemic situation. You must have had some knowledge and experiences about it. Create a HTML and CSS document about the situation, how you handled it, how you spent your time etc.

Activity - 2

Create a webpage of your college / organisation. It should have details of different clubs associated with your workplace and their members. There should be links to register for different events in the clubs where people can register themselves.

Activity - 3

Create your CV using HTML and CSS.

Activity - 4

Create a website for your workplace. You can take the data from your respective workplace websites but make sure you have your own ideas of design .Use HTML, CSS.

Activity - 5

Create a CSS tutorial covering most of the topics of the under CSS. You need to use HTML and CSS. The tutorials must not be duplicate to any tutorial in the web.Good layout as well as user friendliness is desired.
