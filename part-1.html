Inline Styles

Inline styles in CSS could be useful for previewing changes instantly or adding CSS rules to only one or two elements. When you don't have access to your . css file, knowing how inline style CSS works can be convenient.

Inline styles are defined within the "style" attribute of the relevant element

Example :

<h1 style="color:navy;text-align:center;">Hello world </h1>

<p style="color:yellow;">Hello world. </p>

Internal Style Sheet

An internal stylesheet holds CSS rules for the page in the head section of the HTML file. The rules only apply to that page, but you can configure CSS classes and IDs that can be used to style multiple elements in the page code.

Example :

<style>
body { background-color: linen; }
h1 { color: maroon; margin-left: 40px; }
</style>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>

External Style Sheet

An external style sheet is a file with a . css extension that contains CSS style definitions for your webpage. It completely separates the CSS styles from the HTML document making it easy to reuse and maintain.

Example :

<link rel="stylesheet" type="text/css" href="mystyle.css">

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

CSS :

body { background-color: lightblue; }
h1 { color: navy; margin-left: 20px; }

Selectors

CSS selectors are used to find the HTML elements you want to style.

Example :

<p>This is a universal selector.</p>

CSS :

* { text-align: center; color: blue; }

Colors

Colors can be of different types in CSS (Background, text, border etc)

Example :

<p style="background-color:Tomato;">Hakuna matata!!!!</p>

<p style="color:MediumSeaGreen;">Hakuna matata!!!!</p>

Background

The background-image property specifies an image to use as the background of an element

The background-attachment property specifies whether the background image should scroll or be fixed (will not scroll with the rest of the page):

Example :

body { background-image: url("paper.png");
background-attachment: fixed; }

Borders

The CSS border properties allow you to specify the style, width, and color of an element's border.

Example :

CSS :
